# Slack

## Reminder
### Example
```
자신에게 리마인더 보내기	/remind me 프레젠테이션 업데이트 10분 후
멤버에게 리마인더 보내기	/remind	@강민석 보고서 제출 5월 30일
채널에 리마인더 보내기	/remind #디자인-팀 주간 미팅 참석 화요일마다

특정 문구를 매일 특정 시간에 보내기 /remind #channel "text" everyday 22:00
```

### 날짜
```
매일 : everyday
평일 : every weekday
```
